# Book Search Engine

## Description 
This application is a Google Books API search engine built with a RESTful API, and refactor it to be a GraphQL API built with Apollo Server.

## Built With
* Javascript
* Node.js
* React
* MongoDB
* Mern Stack
* Express.js
* HTML
* CSS
